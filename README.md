# Bubble Sort Algorithm

Реализация алгоритма пузырьковой сортировки на Python.

## Описание

Пузырьковая сортировка (Bubble Sort) — простой алгоритм сортировки, который многократно проходит по списку, сравнивает соседние элементы и меняет их местами, если они находятся в неправильном порядке. Этот процесс повторяется до тех пор, пока не останется ни одной пары элементов, требующей обмена.

## Особенности

- ✅ Сортировка по возрастанию
- ✅ Оптимизация с флагом `swapped` — раннее завершение при уже отсортированном массиве
- ✅ Не изменяет оригинальный массив (возвращает копию)
- ✅ Подробные комментарии в коде
- ✅ Примеры использования

## Установка

Никаких дополнительных зависимостей не требуется. Достаточно иметь Python 3.x.

## Использование

### Базовое использование

```python
from bubble_sort import bubble_sort

# Пример 1
arr = [64, 34, 25, 12, 22, 11, 90]
sorted_arr = bubble_sort(arr)
print(sorted_arr)  # [11, 12, 22, 25, 34, 64, 90]

# Пример 2
numbers = [5, 2, 8, 1, 9]
result = bubble_sort(numbers)
print(result)  # [1, 2, 5, 8, 9]
```

### Запуск примера

```bash
python bubble_sort.py
```

## Алгоритм

1. Сравнивает соседние элементы массива
2. Если они в неправильном порядке, меняет их местами
3. Повторяет процесс для каждого элемента
4. С каждым проходом наибольший элемент "всплывает" в конец массива
5. Оптимизация: если за проход не было обменов, массив отсортирован

## Сложность

- **Временная сложность:** O(n²) в худшем и среднем случае, O(n) в лучшем случае (уже отсортированный массив)
- **Пространственная сложность:** O(n) (создается копия массива)
