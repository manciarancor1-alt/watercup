# Water Cup Greeting Program

Простая программа-приветствие на Python.

## Описание

Программа запрашивает имя пользователя и выводит персонализированное приветствие.

## Использование

Запустите программу:
```bash
python greet.py
```

Программа попросит ввести ваше имя и выведет приветствие.

## Требования

- Python 3.x
